# 架构设计

## 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户层                                   │
│    CLI / API / Dashboard / Chat Interface                       │
└───────────────────────────┬─────────────────────────────────────┘
                            │
┌───────────────────────────▼─────────────────────────────────────┐
│                    Event Bus (事件总线)                          │
│              发布/订阅模式，解耦组件通信                           │
└───────────────────────────┬─────────────────────────────────────┘
                            │
┌───────────────────────────▼─────────────────────────────────────┐
│                   Planner Agent (总控)                           │
│         任务分解 → Agent调度 → 结果聚合 → 验证 → 学习              │
└───────────────────────────┬─────────────────────────────────────┘
                            │
    ┌───────────────────────┼───────────────────────┐
    │                       │                       │
    ▼                       ▼                       ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Linux Agent │     │  K8s Agent  │     │   DB Agent  │
│   系统运维   │     │   容器运维   │     │  数据库运维  │
└─────────────┘     └─────────────┘     └─────────────┘
    │                       │                       │
    ▼                       ▼                       ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Log Agent  │     │Monitor Agent│     │Security Agent│
│   日志分析   │     │   监控分析   │     │   安全巡检   │
└─────────────┘     └─────────────┘     └─────────────┘
    │                       │                       │
    └───────────────────────┼───────────────────────┘
                            │
┌───────────────────────────▼─────────────────────────────────────┐
│                      MCP Marketplace                            │
│         SSH | K8s | Docker | DB | Cloud | Monitoring            │
└───────────────────────────┬─────────────────────────────────────┘
                            │
┌───────────────────────────┴─────────────────────────────────────┐
│                         基础设施                                  │
│   本地服务器 | 容器 | Kubernetes | AWS | 阿里云 | 腾讯云           │
└─────────────────────────────────────────────────────────────────┘
```

## 核心组件

### Memory System

| 类型 | 说明 | 用途 |
|------|------|------|
| Working Memory | 工作记忆 | 当前会话上下文 |
| Short-term | 短期记忆 | 近期对话历史 |
| Long-term | 长期记忆 | 持久化存储 |
| Semantic | 语义记忆 | 向量检索 |
| Episodic | 情景记忆 | 具体事件记录 |
